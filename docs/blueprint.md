# **App Name**: RK Portfolio

## Core Features:

- Interactive Resume Timeline: Display resume information in an interactive timeline format using Angular components, allowing users to easily navigate through education, experience, and projects.
- Project Showcase with Filtering: Showcase Java, Spring Boot, and Angular projects with detailed descriptions, tech stacks (displayed using Angular Material chips), and links to GitHub repositories. Allow users to filter projects by technology or category using Angular's filtering capabilities.
- Skill Visualization: Visualize technical skills (Java, Spring Boot, Angular, etc.) using interactive charts or graphs (e.g., using ng2-charts) to showcase proficiency levels. The data will be dynamically rendered using Angular.
- Contact Form: Implement a contact form using Angular Reactive Forms for validation and submission. The form will send data to a backend service (Node.js or Spring Boot) for processing.
- Personalized Greeting: Use generative AI to generate a personalized greeting based on the viewer's location or profession. The AI model is given a tool to allow it to select a fitting greeting.
- GitHub Integration: Display recent GitHub activity (commits, pull requests) by fetching data from the GitHub API and displaying it using Angular components.
- LinkedIn Integration: Include a direct link to your LinkedIn profile using an Angular component.

## Style Guidelines:

- Primary color: Dark slate blue (#37474F) to convey professionalism and technical expertise.
- Background color: Light gray (#ECEFF1) for a clean and modern backdrop.
- Accent color: Teal (#26A69A) to highlight key elements and CTAs.
- Headline font: 'Space Grotesk' sans-serif for headlines. Body font: 'Inter' sans-serif for body text. Note: currently only Google Fonts are supported.
- Use simple, professional line icons to represent different sections and skills.
- Maintain a clean, well-structured layout with clear visual hierarchy using Angular Flex Layout or Grid Layout.
- Incorporate subtle, tasteful animations using Angular Animations to enhance user experience.
